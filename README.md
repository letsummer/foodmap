# 트위터 맛집 지도

**트위터(현 X)에서 추천된 맛집을 모아볼 수 있는 지도**

(링크: https://foodmap-hdef.onrender.com/)


## 개발 동기
**트위터(현 X)에 올라온 맛집이 가장 신뢰도가 높다고 판단하여 북마크를 하여 저장하지만 막상 찾으려면 힘들어서 만들게 됨.**
